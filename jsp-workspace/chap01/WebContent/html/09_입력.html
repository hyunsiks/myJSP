# 사용자의 선택을 전송하기 위한 절차

  1. 모든 데이터는 <Form>단위로 전송되어야 한다
  2. 해당 from에서 submit이 발생하면 입력된 모든 데이터를 전송한다
  3. 전송하고자 하는 데이터에는 반드시 name 속성이 있어야 한다
  4. 데이터를 전송할 때 name속성이 되고 value 속성이 값이 된다
  5. GET방식으로 전송하면 URL뒤에 데이터가 붙어서 전송된다 http://host:port/uri?name1=value1&name2=value2
  6. 그 외의 방식은 데이터가 주소에 노출되지 않는다
